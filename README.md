# Recipy app with GraphQL

Learning graphQL basics.

# Install

To install server side dependencies, run
```
cd server
npm install
```
To install client side dependencies, run
```
cd client
npm install
```
# Use

Fetch data from a local JSON file.
To run the server:
```
cd server
node src/index.js
```
To run the client:
```
cd client
npm start
```
# Built With

- apollo-client
- apollo-server
